## Local Development

1. Install dependencies:

   ```bash
   pnpm i
   ```

2. Navigate to the frontend directory:

   ```bash
   cd fe
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

4. Open the URL shown in your terminal to start developing.
